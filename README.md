# Premium Scripts 精品脚本集

一个持续更新的 Python 脚本合集，涵盖文件处理、数据转换、自动化、报告生成等多种场景。

## 脚本列表

### 文件工具
| 脚本 | 功能 |
|------|------|
| pdf_toolkit.py | PDF 合并、拆分、水印 |
| pdf_merge.py / pdf_merger.py | PDF 文件合并 |
| pdf_generator.py | 从文本生成PDF |
| batch_image_compressor.py | 批量图片压缩 |
| image_resizer.py | 图片批量缩放 |
| image_watermarker.py | 图片批量水印 |
| file_organizer.py | 按类型自动整理文件 |
| batch_rename.py / batch_renamer.py | 批量重命名 |
| smart_sync_backup.py | 智能同步备份 |
| watermarker.py | 文件水印工具 |

### 数据处理
| 脚本 | 功能 |
|------|------|
| data_converter.py | 多种数据格式互转 |
| data_cleaner.py / excel_clean.py | 数据清洗 |
| csv2xlsx.py / csv_to_excel.py | CSV 转 Excel |
| csv_merger.py | CSV 文件合并 |
| excel_dashboard_generator.py | 自动生成Excel仪表盘 |
| excel_report_automation.py | Excel 报表自动生成 |
| excel_reporter.py | Excel 报告生成 |
| excel_cleaner.py | Excel 数据清洗 |

### 网络监控
| 脚本 | 功能 |
|------|------|
| web_monitor_alert.py | 网页监控+变动告警 |
| web_monitor.py / website_monitor.py | 网站可用性监控 |
| web_scraper.py / scraper.py | 网页数据采集 |
| email_campaign.py / email_sender.py | 邮件群发 |
| youtube_dl_gui.py | YouTube 下载GUI |
| api_tester.py | API 接口测试工具 |
| qr_generator.py | 二维码生成器 |

### 文案工具
| 脚本 | 功能 |
|------|------|
| youtube_dl_gui.py | YouTube 视频下载 |
| email_campaign.py | 邮件营销 |

## 使用

```bash
git clone https://github.com/SKYJAMES777/premium-scripts.git
cd premium-scripts
python pdf_toolkit.py
```
